# Installation

## Requirements

* Windows 10 or Windows 11
* Gothic Remake installed
* Administrator privileges (recommended)

## Installation Steps

1. Download the latest release from the Releases page.
2. Extract the ZIP archive.
3. Run `GothicRemakeMod.exe`.
4. Select your Gothic Remake installation folder.
5. Click **Install**.
6. Wait for the installation process to complete.
7. Launch the game and enjoy the mod.

## Updating

1. Download the latest version.
2. Run the installer.
3. Select the existing game directory.
4. Complete the update process.

## Uninstallation

1. Run the installer again.
2. Select **Uninstall**.
3. Restore original game files if necessary.
