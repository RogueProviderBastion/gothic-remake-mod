# Frequently Asked Questions

## Does this mod affect save files?

No. The mod is designed to work without modifying save files.

## Can I uninstall the mod?

Yes. The installer supports safe removal.

## Is this mod compatible with future game updates?

Compatibility may vary depending on game updates. Check Releases for updated versions.

## Does the mod impact performance?

The mod is designed to remain lightweight and stable.

## Is an internet connection required?

No. Installation and usage work offline.

## Is this an official Gothic Remake modification?

No. This is a community-made project.
