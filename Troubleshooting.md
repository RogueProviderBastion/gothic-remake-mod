# Troubleshooting

## Game Does Not Launch

- Verify game files.
- Reinstall the mod.
- Ensure the correct installation path was selected.

## Installer Cannot Find the Game

- Manually select the Gothic Remake folder.
- Make sure the game is installed correctly.

## Performance Issues

- Disable conflicting modifications.
- Update graphics drivers.
- Restart the game after installation.

## Need More Help?

Create an Issue on GitHub and include:

- Game version
- Mod version
- Operating system
- Error description
